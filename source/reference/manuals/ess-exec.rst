--------------------------------
**ess exec**
--------------------------------

::

    usage: ess exec [-h] [--master] [--debug] [--s3out S3OUT] command
    
    Execute arbitrary command
    
    positional arguments:
      command        Filter to use
    
    optional arguments:
      -h, --help     show this help message and exit
      --master       where to run
      --debug        debug mode
      --s3out S3OUT  send output to an s3 bucket
