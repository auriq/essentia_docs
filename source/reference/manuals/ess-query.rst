--------------------------------
**ess query**
--------------------------------

::

    usage: ess query [-h] [--label LABEL] [--check] command [command ...]
    
    SQL-like command.
    
    positional arguments:
      command        SQL command
    
    optional arguments:
      -h, --help     show this help message and exit
      --label LABEL  Specify the datastore to use
      --check        check SQL syntax only
